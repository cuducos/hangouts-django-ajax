Endpoints

Browser navigation

If user is authenticated (use Django Admin) it shows the form to make a new booking via AJAX (to be created during the Hangout).

  • /bookings/: list all bookings from the current month
  • /bookings/<year>/<month>/ (e.g. /bookings/2016/12/ ): list all bookings from the current month

API

All endpoints requires authentication (use Django Admin).

  • GET /api/bookings/: list all bookings
  • POST /api/bookings/: create new booking
